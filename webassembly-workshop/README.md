# 🚀 WebAssembly (WASM) in Cloud-Native Environments
### A 4-Hour Hands-On Workshop

> **The Better Software Initiative** | Training conducted by an experienced consultant

---

## 🎯 Workshop Overview

WebAssembly is emerging as a powerful runtime for cloud-native, edge, and microservice architectures. This workshop explores WASM as a **secure, fast alternative to containers**, giving you practical skills to build and evaluate WASM workloads in real-world cloud-native contexts.

**Duration:** 4 hours  
**Language:** English  
**Format:** Theory + Hands-on Labs

By the end of this workshop you will be able to:

- ✅ Explain the WASM execution model and why it matters for cloud-native
